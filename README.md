# STOA60

Website for **STOA60**, a self-managed underground music venue in Heraklion, Crete. Running since 2015 — hundreds of gigs and counting.

## What it does
- **Next gig** — hero section showing the upcoming show poster
- **Gig mail alert** — newsletter signup for show announcements
- **About** — the venue's story and values
- **Poster archive** — 189 posters across 9 seasons (2015–2026), with lightbox viewer

## Tech stack
- [Astro](https://astro.build) 5.x (static site)
- Tailwind CSS v4
- TypeScript
- Deployed on Cloudflare Pages

## Development

```bash
npm install          # install dependencies
npm run dev          # dev server at localhost:4321
npm run build        # production build → ./dist/
npm run preview      # preview production build locally
```
